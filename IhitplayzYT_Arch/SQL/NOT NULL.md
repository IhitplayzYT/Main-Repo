##### NOT NULL
Implies that the following column can't be NULL / Empty.

##### Usage:
eg:-
```sql
CREATE TABLE Example(
Name varchar(10) NOT NULL,  -- Here name can't be Empty
Age int   -- Age can be NULL or Empty
);
```
