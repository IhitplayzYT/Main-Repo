# Stdoslib

### A functional paradigm based library for C containing many convenient macros and functions

# How to Install ?

1. Clone the Repository locally on your device
```bash     
git clone https://github.com/IhitplayzYT/Stdoslib.git
```

2. Change to cloned directory
```bash 
cd Stdoslib
```

3. Install by using make
```bash
sudo make install
```

# How to uninstall ?
1. Change to project directory
```bash 
cd Stdoslib
```

2. Uninstall by using make
```bash
sudo make uninstall
```

# Usage 
#### Simply add lib include in the C project
```c
#include <stdoslib.h>

<Your code goes here>

```

# Linking library
### To Link the library use the -lstdoslib flag during compilation
