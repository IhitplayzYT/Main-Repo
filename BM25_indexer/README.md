# BM25 Indexer 

## Features
- Stopword Removal
- Snowball Stemming
- Lanchaster Stemming
- Sorting files by BM25 index of File
- Reactive TUI for real time search


# Install:

## 1.Compile the executable & Use
1. Clone the Repo
```bash
git clone https://github.com/IhitplayzYT/BM_25-indexer.git
```
2. Change directory to the Repo
```bash
cd BM_25-indexer
```
3. Build the executable
```bash
make
```
4. Use the executable. Start with ./BM25_indexer -h


## 2.Install executable to your Binaries
1. Clone the Repo
```bash
git clone https://github.com/IhitplayzYT/BM_25-indexer.git
```
2. Change directory to the Repo
```bash
cd BM_25-indexer
```
3. Install the executable, Sudo required for adding executable to /usr/bin
```bash
sudo make install
```


## 3. Deleting the executable
1. Change directory to Repo
```bash
cd BM_25-indexer
```
2.Run the Uninstall rule, Sudo to remove the executable from /usr/bin
```bash
sudo make uninstall
```
